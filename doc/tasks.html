Automated Analysis Quickstart Guide

Load-up a brand new origin instance and run each command in order. Pay attention to what you're doing, and by the end you'll know how to use the new tools new data analysis workflow

Gain Function

>> sc gain loads an example gain function ABF. Similar commands exist for ramp, iv, mt, drug, etc.

enable event detection with settings to detect APs.

>> sc auto The goal is to have an automated routine for every primary type of ABF. Note that default workbook names (EventsEp, MemTests, etc) are *ALWAYS* deleted automatically.

 >> sc gain 1; sc auto; sc gain 2; sc auto; sc gain 3; sc auto; sc gain 4; sc auto; sc gain 5; sc auto; sc gain 6; sc auto; sc gain 7; sc auto; sc gain 8; sc auto; sc gain 9; sc auto; sc gain 10; sc auto; sc gain 11; sc auto; sc gain 12; sc auto; sc gain 13; sc auto; sc gain 14; sc auto; sc gain 15; sc auto; Note how output sheets are PARENTID_ABFID_STEP. This makes getcols easy.

>> sc getcols _step1 command Inst This needs to have the worksheet selected. It's an improved getcols does string matching, and if given 2 arguments provides XY pairs. (see 'sc docs' for even more usage features)

>> sc replace nan 0 It's obvious what this does, and what cases it's useful (e.g., 0 is good for avg freq, nan is good for avg amplitude)
