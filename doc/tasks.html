AP Gain

>> sc fsi loads an example ABF from a fast-spiking interneuron

enable event detection with settings to detect APs.

>> sc auto automatically determine how to analyze this file and do it. In this case, it's a gain function, so a single output workbook will be made or added to. Any gain-related analysis will be added to this workbook.

Note that sheet names are PARENT_ABF_STEP. No sheet of this workbook should ever be manually edited.

Note also that re-running the same "sc auto" analysis is fine. The sheet will be overwritten, and all is well.

>> sc fsi 1; sc auto; sc fsi 2; sc auto; sc fsi 3; sc auto; sc fsi 4; sc auto; sc fsi 5; sc auto; sc fsi 6; sc auto; sc fsi 7; sc auto; sc fsi 8; sc auto; sc fsi 9; sc auto; sc fsi 10; sc auto; sc fsi 11; sc auto; sc fsi 12; sc auto; sc fsi 13; sc auto; sc fsi 14; sc auto; sc fsi 15; sc auto; Let's do the same thing a bunch of times in a row.

>> sc replace nan 0 force sweeps with no APs to report 0. This is good for Steady State Freq. This is not desired for AvgAmp (for example).
