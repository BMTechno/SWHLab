Manual Installation
1. Extract pythonModules.zip into C:/Apps/pythonModules/
2. Extract the most recent version of SWHLab into C:/Apps/pythonModules/swhlab/

Origin (CJFLab) Installation
Simply run the origin command pyload.
If SWHLab breaks (crashes on load), delete C:/Apps/pythonModules/swhlab/ and try again.

Integrating with your system python
This is not required. I do it so I can write bash scripts to run python and import swhlab from anywhere.
Spyder documentation works on PyOrigin and SWHLab if you do this too.
Set the following windows environment variables:
PYTHONPATH = C:/Apps/pythonModules
PATH = C:/python/install/folder (where python.exe resides, can be any python distro)
